# 音声ファーストナビゲーションコンセプト

## 概要 
このリポジトリは、**音声を主軸としたナビゲーションおよびガイダンスの設計思想と構造**を共有するためのコンセプトリポジトリです。

画面を見ることを前提とせず、  
**「聞く・話す・確認する」**という人間の自然な行動を中心に設計します。

This repository shares the **design philosophy and structural concept** of a navigation and guidance system that places **voice as the primary interface**.

## なぜ音声ファースト・ナビゲーションなのか

本コンセプトは、画面操作を前提とした従来のUI設計に対する問題意識から生まれています。

現代のアプリケーションやナビゲーションは、多くの場合「見る」「読む」「タップする」ことを前提に設計されています。しかし、人間の本来の行動は必ずしも画面注視を伴うものではありません。

歩いているとき、運転しているとき、作業中、料理中、介護中、視覚に制限がある状況──  
そうした場面において、画面中心のUIは人間の行動と衝突します。

人は本来、
- 話しかける  
- 聞く  
- 確認する  

という行為を、極めて自然に行っています。

本コンセプトでは、この「人間の自然な行動」を起点に据え、  
**音声を主（ファースト）とし、画面を補助とするナビゲーション設計思想**を提示します。

これは特定の実装や技術を縛るものではありません。  
思想・構造・設計の方向性そのものを共有するための、概念リファレンスです。

本リポジトリは、
- 実装コードの公開を目的としません  
- 特定企業・製品への帰属を主張しません  

ただし、この思想が  
**誰かの実装・誰かの製品・誰かの社会実装につながること**  
それ自体を強く意図しています。

音声ファースト・ナビゲーションは、  
高齢者、子ども、視覚制約のある人、作業中の人、移動中の人を含め、  
「使える人を増やす」ための設計思想です。

この考え方が、広く共有され、各所で発展していくことを願っています。
This concept originated from the ideas and design philosophy
of Tenchi-Ai Project / Ojichan Konbini.

This repository does not assert implementation ownership.
It exists to share conceptual and design perspectives.
## このコンセプトが目指すもの 

- ユーザーが画面を見ていることを前提にしない  
- 音声による誘導・確認・判断を中心とした体験設計  
- ハンズフリー、多言語、アクセシビリティへの配慮  
- 移動中・作業中・視線を使えない状況でも成立するナビゲーション
